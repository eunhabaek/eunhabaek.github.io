## **1. XHTML 기본 구조** ##
```css
<!DOCTYPE html>
<html lang="en">
    <head>
        <!--브라우저에 전달되는 내용-->
    <title>Document</title><!--문서 제목으로 탭 네임-->
</head>
<body>
    <!--웹페이지에 출력되는 부분-->
</body>
</html>
```

## **2. 공통 속성 종류** ##

- id: 페이지 내 구분 위함, 중복 X
- class: 묶음을 만들기 위함, 여러개 포함
- name: 서버에 데이터 전달 위해 사용, 서버가 읽음, 중복 O
- data: 태그 안에 데이터 포함 위해 사용
